Important: Chrome and Edge require additional setup for Tampermonkey to inject scripts. Without this, the extension installs but scripts won't run.

Chrome / Edge (v138+)

  1. Right-click the Tampermonkey icon in your toolbar
  2. Select "Manage Extension"
  3. Enable the "Allow User Scripts" toggle

Chrome / Edge (older versions)

  1. Go to chrome://extensions or edge://extensions
  2. Enable Developer Mode (toggle in top-right corner)

Firefox / Safari

No additional setup needed. Scripts run automatically after installation.
