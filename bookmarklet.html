Note: The bookmarklet runs once per click. If you navigate away and come back, just click it again. For auto-run, use Tampermonkey (see below).

How to read it

  • Bar behind reticle ‚Üí Under budget, you have capacity
  • Bar ahead of reticle ‚Üí Over budget, slow down
  • Bar at reticle ‚Üí Perfect pacing

For auto-run (Tampermonkey)

If you want it to run automatically every time you visit the page:

  1. Install Tampermonkey
  2. Click the Tampermonkey icon ‚Üí "Create new script"
  3. Paste the full script from usage-reticle.user.js
  4. Save (Ctrl+S)
